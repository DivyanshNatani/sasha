About Sasha

Name of bot on Telgram: Team_Automated_Bot
The Internet of Things has been one of the hot current topics along with Artificial Intelligence and Machine Learning. So combining ideas from all these fields, our plan is to make a ChatBot with personal assistance.Almost any function which involves interaction with a customer can be done by a Chatbot.

Our Project has successfully made a chat bot which can be used for controlling home appliances such as lights, fans, etc and would be very efficient to add many devices further on. Along with this it is multi- functional and can do various other things apart from Home Automation. It can tell current weather detail of 500+ cities, can tell jokes, trending news etc.

For mapping the live status of devices, we have created a dashboard, which is suitable substitute of hardware for prototype softwere. We have also created a Home Setup Code Generator(HSCG) portal which will help user to generate code to integrate your home to our bot.
