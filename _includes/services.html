Name of bot on Telgram: Team_Automated_Bot

The Internet of Things has been one of the hot current topics along with Artificial Intelligence and Machine Learning. So combining ideas from all these fields, our plan is to make a ChatBot with personal assistance. Almost any function which involves interaction with a customer can be done by a Chatbot.
Our Project is to successfully make a chat bot which can be used for controlling home appliances such as lights, fans, etc and would be very efficient to add many devices further on. Along with this we were trying to make a chatbot which would be multi- functional and do various things other than Home Automation. After the creation of Chatbot, we created a dashboard which would map to live status of devices.
